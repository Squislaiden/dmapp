---- Changelog (version 1.1.1): Visual overhaul.
• New Spells List page added, accessible via "~Tables~" Tab.
• New Items List and Feats pages accessible via the "~Tables~ Tab" for my WIP Equipment Overhaul Homebrew. Currently Empty.
• Tons of code cleanup and more optimized storage for randomization and data functions.
• Basic decks have been added: Standard Deck, w/ Jokers, Tarot Minor/Major/Full, and Deck of Beasts (original and expanded).
• Utilization of Cookies and LocalStorage to remember.
   • The Dice Roller tracks the last 50 rolls and can be viewed with the History button.
   • The Deck Status button allows you to view which cards have been drawn from that deck and which cards are still in it. The Reset Deck button will clear saved data for the selected deck.
