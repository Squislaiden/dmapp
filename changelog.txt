---- Changelog (version 1.1.1): Visual overhaul.
• New Spells List page added, accessible via "~Tables~" Tab. Some tweaks have been done:
   • Spells have a Mana Cost and type. Spell Levels are still displayed.
   • Wording changes: Poisoned --> Debilitated, Exhaustion --> Malaise.
   • AC is now Dodge. Some AC spells will grant Block.
   • Resistance has 5 stages (25%/50%/75% less taken, and immunity at 5). Stage 4 grants no benefit.
• Overhaul Items List and Feats pages accessible via the "~Tables~ Tab" for my WIP Equipment Overhaul Homebrew. Currently Empty.
• Tons of code cleanup and more optimized storage for randomization and data functions.
• Added Gender and Sexuality randomizers to Character Tab.
• All Decks added.
• Jewelry Tab and Random Enemy Generator now fully work in the new page. The only function that still requires the old page is the Glyphs tab.
• Utilization of Cookies and LocalStorage to remember.
   • The Dice Roller tracks the last 50 rolls and can be viewed with the History button.
   • The Deck History button will show the last 50 cards drawn and their respective deck.
   • The Deck Status button allows you to view which cards have been drawn from that deck and which cards are still in it.
   • The Reset Deck button will clear saved data for the selected deck. Reset All Decks will clear all saved data for all decks.
